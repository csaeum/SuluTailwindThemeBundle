## 📝 Installation

### Composer

```bash
composer require itech-world/sulu-theme-bundle
```

### Symfony Flex

If you don't use Symfony Flex, you can add the bundle to your `config/bundles.php` file:

```php
return [
    // ...
    ItechWorld\SuluThemeBundle\ItechWorldSuluThemeBundle::class => ['all' => true],
];
```
